# Security policy

## Reporting a vulnerability

If you discover a security vulnerability in walastack-rs, please report it
privately by emailing **contact@walalabs.tech**.

Do **not** open a public GitHub issue for security-sensitive reports.

We will acknowledge receipt within 72 hours and aim to provide an initial
assessment within one week.

## Supported versions

WalaStack is pre-release. Only the `main` branch and the latest published
pre-release receive security updates.

## Disclosure

Once a fix is available, the vulnerability will be disclosed in the
[CHANGELOG](CHANGELOG.md) and via a GitHub Security Advisory.
